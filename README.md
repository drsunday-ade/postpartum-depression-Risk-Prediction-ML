# Postpartum Depression Risk Prediction (Bangladesh) — ML Pipeline

Reproducible analysis code for a prediction-model development study with internal validation using a de-identified postpartum dataset from Bangladesh (women within 24 months postpartum). The repository supports transparent evaluation of machine-learning approaches for postpartum depression (PPD) risk prediction.

Public dataset (not redistributed here):
